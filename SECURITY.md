# Security Policy

## Supported Versions

The following versions of Impact Tracker are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open an issue. Instead, please report it to the maintainer directly at the contact email provided in the repository profile.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
